# Arquitectura-de-Procesamiento-de-Video-con-Patr-n-Fan-Out
Implementación de una arquitectura de procesamiento de video basada en eventos, desplegada mediante Infrastructure as Code (IaC) con Terraform y simulada localmente con LocalStack. El objetivo central del sistema es procesar un video subido por un usuario y generar automáticamente múltiples versiones en distintas resoluciones.
